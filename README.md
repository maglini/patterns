Создание нового экземпляра продукта возможно только через обращение к фабрике.
Различия: Abstract Factory производит группы продуктов, а Factory Method производит одиночный конкретный тип продукта.

Отмечу, что при всех достоинствах этих двух шаблонов, у них есть и недостатки. Например, в случае Factory Method, необходимо создавать отдельную фабрику под каждый новый тип продукта. А в случае Abstract Factory, неудобства наступают при возникновении необходимости расширения ассортимента производимых фабрикой продуктов.
